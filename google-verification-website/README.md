# CheshireCode OpenClaw Google Integration Verification Website

This website is created for Google verification purposes to support the Google OAuth integration in cheshirecode's personal OpenClaw integration project.

## Purpose

This site serves as the application domain for Google OAuth verification for cheshirecode's Google integration tools that work with the OpenClaw ecosystem. It includes:

- Application home page
- Privacy policy page
- Terms of service page

## Deployment

Deploy to Vercel using:

```bash
cd google-verification-website
vercel --prod
```

## Pages

- Home: `/` - Main landing page describing the Google integration service
- Privacy Policy: `/privacy` - Detailed privacy policy for data handling
- Terms of Service: `/terms` - Terms governing the use of the service

## Google Verification Requirements

This website meets Google's verification requirements:
- Clear application branding and description
- Public privacy policy
- Public terms of service
- Professional appearance and content

## About the Project

This is a personal project by GitHub user cheshirecode designed to integrate Google services with the OpenClaw ecosystem for enhanced AI assistant capabilities.
