# Moltbot Helpers Website

This website serves as the public-facing site for Moltbot Helpers, primarily created to satisfy Google's OAuth application verification requirements.

## Purpose

This site provides:
- Public homepage for the Moltbot Helpers application
- Privacy Policy page
- Terms of Service page
- Branding information for Google verification

## Deployment

This site is deployed on Vercel at: https://moltbot-helpers.vercel.app

## Google OAuth Verification

This website provides the required:
- Application home page
- Privacy Policy
- Terms of Service
- Branding information

All necessary for Google OAuth application verification.
